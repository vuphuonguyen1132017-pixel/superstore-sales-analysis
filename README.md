# superstore-sales-analysis

## Project Overview

This project analyzes retail sales data using Python to discover businnes insights.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Business Questions

- How do sales change over time?
- Which region has the highest sales?
- Does discount affect profit?
- Who are the top customers?

## Conclusion

This project provides insights into sales trends, profitability, and customer behavior to support businnes decisions.
